### Диаграмма прогноза

По вертикали значение `100%` означает полное распределение категорий бюджета,
по горизонтали — дни текущего месяца. Диаграмма разделена на две секции:
- **Исторические данные**: этот раздел показывает накопленные расходы с течением времени.
- **Прогноз**: этот раздел предсказывает расходы с помощью моделирования Монте-Карло.
