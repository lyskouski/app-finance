**Fingrom** — приложение для финансового учёта с открытым исходным кодом, кроссплатформенное, без рекламы и ограничений.
Цель решения — создать интуитивное, эффективное и инклюзивное приложение для финансового учёта.
Оно позволяет пользователям без усилий управлять своими финансами, гарантируя, что никто не останется позади.

[![Смотреть видео](../images/presentation_en.png)](https://youtu.be/sNTbpILLsOw)

### Функциональность
- Учёт (тип счёта, валюта/криптовалюта)
  - Простая группировка с помощью символа `/` (в названии) на главной странице
  - Сортировка по суммам, датам и названиям
  - Журнал транзакций
  - Заморозка суммы по дате обновления (для импорта предыдущей истории)
- Категории бюджета
  - Простая группировка с помощью символа `/` (в названии) на главной странице
  - Сортировка по суммам, датам и названиям
  - С перерасчётом лимитов:
    - Обновляются в начале каждого месяца
    - Настраиваемые лимиты в месяц
    - Относительно (0,0 ... 1,0) к доходу
  - Или без ограничений, показывая израсходованную сумму
  - Разные временные интервалы: еженедельно, ежемесячно, ежегодно
  - Настраиваемый день начала недели и месяца
  - Сводка по бюджету: опция игнорирования отрицательных значений
- Чеки, переводы, доходы (счета-фактуры)
  - Повторяющиеся платежи (с домашним виджетом)
  - Фильтрация
  - Предположение/прогноз категории бюджета
  - Разделение чека
- Определение целей
- Курсы обмена, валюта по умолчанию для итогов
- Метрики:
  - Бюджет:
    - Прогноз (с моделированием Монте-Карло)
    - Лимит бюджета и расходы в месяц
  - Счёт:
    - Свечная диаграмма (OHLC)
    - Радар здоровья доходов
    - Распределение валют
  - Счета:
    - Расходы с начала года
    - Гонка расходов по категориям
  - Диаграмма целей
  - Исторический график валюты
- Синхронизация между устройствами (P2P)
- Восстановление через WebDav или файл
- Импорт из файлов `CSV`, `QIF`, `OFX` для счетов и счетов-фактур
- Экспорт в файл Excel `XLSX`
- Шифрование данных
- Локализация
- Безопасность
  - Защита одноразовым кодом (2FA)
  - Код восстановления
  - Биометрическая аутентификация
- Пользовательский опыт
  - Настраиваемая главная страница (несколько конфигураций для каждого параметра `ширина х высота`)
  - Адаптивный дизайн
    - Адаптивная панель навигации (сверху, снизу, справа) и вкладки (сверху, слева)
  - Режим темы (тёмный, светлый, системный) с определением палитры (системная, пользовательская, личная — селектор цветов)
  - Сохранение последнего выбора для счёта, бюджета и валюты
  - Автопрокрутка к сфокусированному элементу на форме
  - Развёртывание / свёртывание разделов на главной странице
  - Быстрый доступ к действиям редактирования и удаления через свайп
  - Увеличение/уменьшение масштаба (от 60% до 200%) через «Параметры»
  - Ярлыки
